Carregue a sessaoSelecione o .json exportado pelo bookmarklet PARAR.
2

3
Explore as abasRequisicoes, Console, Cookies, Storage, e importe relatorios em Seguranca e Performance.
Como importar relatorios (OWASP ZAP, PageSpeed, Lighthouse)?

OWASP ZAP (aba Seguranca): no ZAP, gere Report → Generate Report em HTML (ou JSON) e importe no botao "Importar relatorio OWASP ZAP". Os alertas entram no resumo consolidado de severidades.

Lighthouse do DevTools (aba Performance): no Chrome DevTools → aba Lighthouse → Analyze → menu Export → Save as JSON. Importe no botao "Importar JSON do Lighthouse (DevTools)" para ver scores, filmstrip, screenshot e treemap.

PageSpeed Insights (aba Performance): rode em pagespeed.web.dev (ou pelo botao "Rodar no Lighthouse"), salve a pagina do resultado (Ctrl/Cmd+S, "completa") e importe o .htm no botao "Importar HTML do PageSpeed".
